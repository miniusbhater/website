GetThatOS is a simple tool to download operating system disk images. It supports Windows and macOS downloads. The project is fully open source and free on GitHub.

Download Source

GetThatOS is free and always will be. If you paid for it you got scammed.


Things to know:

See a cmd window pop up when you open GetThatOS? Don't worry! This is just GetThatIntegrity doing it's job. It checks to make sure all the files required for GetThatOS exist and if they don't it will prompt you to reinstall to help reduce the risk of any issues using GetThatOS. It is also open source just like GetThatOS and is located in the GetThatOS Github Repo. It will be located at "C:\Program Files (x86)\GetThatOS\GetThatIntegrity.exe" on your pc.




All disk images are known to be safe and not tampered with. This includes the images sourced from archive.org. All images on GetThatOS have been tested to ensure they work without issue.
